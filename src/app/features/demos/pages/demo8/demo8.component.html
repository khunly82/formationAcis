Les pipes sont des fonctions permettant de transformer une valeur.
String en minuscule:
{{ 'HELLO WORLD'|lowercase }}
String en majuscule:
{{ 'HELLO WORLD'|uppercase }}
String 1ères lettres en majuscule:
{{ 'HELLO WORLD'|titlecase }}
Number non formaté:
{{ 1.1 * 3 }}
Number formatté:
{{ 1.1 * 3|number: '3.0-2' }}
Montant:
{{ 1.1 * 3|currency: 'EUR' }}
Date non formatée:
{{ Date() }}
Date formattée:
{{ Date()|date: 'dd MMMM yyyy HH:mm' }}
