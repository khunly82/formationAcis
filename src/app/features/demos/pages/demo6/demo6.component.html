Permet de définir un template qui ne sera pas «Render» par défaut dans un composant.

Ce template n'est pas affiché par défaut

On peut alors utilser une directives structurelles pout l'afficher (*ngIf, *ngFor, *ngTemplateOutlet, ...)

À différents endroits

On peut fournir un context à un template

Ce template affichera des valeurs différentes selon le contexte:

Hello {{ context }} !!

On peut alors fournir des données différentes au template

 Remarque: <ng-container></ng-container> est souvent utilisé avec des directives structurelles pour ne pas créer de balise englobante.
